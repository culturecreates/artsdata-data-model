Person

Schema.org definition: A person (alive, dead, undead, or fictional).

Subclass Of: http://schema.org/Thing

Adding people

Here is a preliminary document (French only) on a template to add People.

Another way is to add a performing arts person in Wikidata and fill in the following required fields. Please add as much data as possible to help cases that require disambiguation. We recommend linking social media profiles, and external global IDs like ISNI, but these are not mandatory.

  • Given name
  • Family name
  • Occupation
    • actor
    • stage director
    • musician
  • Instance of human
  • Gender
    • male
    • female
    • intersex
    • transgender female
    • transgender male
    • non-binary
  • Date of birth

ShEx Validator

Artsdata uses ShEx to validate data for import. If the data passes the ShEx validation, the person will automatically be added to Artsdata.ca in a couple of days.
