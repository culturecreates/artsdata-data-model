# Vocabulaire contrôlé Artsdata des types de lieux

| <!-- -->            | <!-- -->                                                                    |
| ------------------- | --------------------------------------------------------------------------- |
| Système de concepts | [adr:ArtsdataPlaceTypes](http://kg.artsdata.ca/resource/ArtsdataPlaceTypes) |
| Version             | 1.0.1                                                                       |
| Date de publication | Février 2026                                                                |

Cette page présente le vocabulaire contrôlé utilisé dans le graphe de connaissances Artsdata pour décrire les types de lieux dans le secteur des arts. Il classe les lieux selon leur conception, leur fonction et leur portée. Il établit notamment une distinction entre les bâtiments entiers et les espaces intérieurs situés à l’intérieur de bâtiments.

Ce vocabulaire contrôlé s’inspire largement de la typologie du [WikiProjet Wikidata sur les lieux culturels](https://wikidata.org/wiki/Wikidata:WikiProject_Cultural_venues/Typology). Il importe des concepts de Wikidata, de l’[Art and Architecture Thesaurus](https://www.getty.edu/research/tools/vocabularies/aat/), de [Nomenclature](https://page.nomenclature.info/apropos-about.app) et de [Schema.org](https://schema.org/). Le concept [adr:PerformanceHall](http://kg.artsdata.ca/resource/PerformanceHall) est équivalent à la [classe Room](https://documentation.datascene.ca/references/room/) dans Datascène.

### Utilisation

Le vocabulaire contrôlé Artsdata des types de lieux peut être utilisé avec la propriété `schema:additionalType`, en combinaison avec la classe générique `schema:Place` comme type principal. Ainsi, une entité de lieu dans Artsdata peut avoir à la fois le type `schema:Place` et un type additionnel provenant du vocabulaire contrôlé Artsdata des types de lieux, au moyen de la propriété `schema:additionalType`.

Voici un exemple en JSON-LD :

```
"@context":"http://schema.org",
"@type": "Place",
"additionalType": "http://kg.artsdata.ca/resource/PerformanceHall"
```
