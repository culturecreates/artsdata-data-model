Artsdata.ca Website

Here is the website of Artsdata.ca. The website provides a YASGUI user interface for writing SPARQL, and general info about Artsdata. This website is also a Lightweight Linked Data Server and Proxy that performs dereferencing for resources stored in the knowledge graph.
